# 参数配置

## LContent Attributes
| 参数    | 说明     | 类型    | 可选值      | 默认值 |
|---------|----------|---------|-------------|--------|
| formConfig | [LForm Attributes](/guide/l-form/options.md#lform-attributes) | object | - | - |
| tableConfig | [LTable Attributes](/guide/l-table/options.md#ltable-attributes) | object| - |-|

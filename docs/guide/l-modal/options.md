# 参数配置

## LFormModal Attributes
| 参数    | 说明     | 类型    | 可选值      | 默认值 |
|---------|----------|---------|-------------|--------|
| show | 显示弹出，与 `.sync` 一起使用 | boolean| - |-|
| submit | 表单提交方法 | ()=>{} | - | - |
| 其他参数 | 与 [el-dialog](https://element.eleme.cn/#/zh-CN/component/dialog#attributes) 相同 | - | - | - |
